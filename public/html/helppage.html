Help Page

This Helppage will explain how to work with the Netfloc GUI. Below there is a link
that will link you to the Documentation of Netfloc itself to give you an overview about Netfloc.

http://netfloc.readthedocs.io/en/latest/

New Features of the GUI:

The GUI offers now the function to create SFCs made out of VNFs as well as VNFs made out of Neutron Ports.
The GUI also offers a function to delete one or multiple already created SFCs or VNFs with the delete button.
The GUI also offers a new refresh funktion.

How to for Service Function Chaining

  • Open page to list all the already created SFCs
  • Choose one or more SFCs with the checkbox function
  • Click delete with the button in the right upper corner to delete the SFCs
  • Click the create button in the right upper corner to open the pop-up for creating SFCs
  • Choose the number of VNFs you want to chain together
  • Order the VNFs with the order function in the left corner of the table
  • Click create SFC to create a SFC and automatically close the pop-up
  • Click the refresh button in the upper right corner to refresh the page

How to for Virtual Network Function

  • Open page to list all the already created VNFs
  • Choose one or more VNFs with the checkbox function
  • Click delete with the button in the right upper corner to delete the VNFs
  • Click the create button in the right upper corner to open the pop-up for creating VNFs
  • Choose a name and a description for the VNF you want to create
  • Choose a ingress port and an egress port in the first row of the table
  • Click create VNF to create a VNF and automatically close the pop-up
  • Click the refresh button in the upper right corner to refresh the page
